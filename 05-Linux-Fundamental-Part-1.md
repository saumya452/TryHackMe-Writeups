# Linux Fundamentals Part 1

## Summary

I learned the basics of Linux and how to use the command line. This room introduced essential Linux commands used for navigating directories, managing files, and working efficiently in a Linux environment.

## Key Learnings

- Learned how to navigate the Linux file system
- Used commands such as pwd, ls, cd, cat, mkdir, touch, cp, mv, and rm
- Understood the difference between absolute and relative paths
- Learned how to view and manage files from the terminal
- Gained hands-on experience using Linux commands
