# Vulnversity

## Summary

I completed the Vulnversity room on TryHackMe, a beginner-friendly CTF focused on enumeration, web exploitation, and privilege escalation. The room provided hands-on experience with reconnaissance techniques, discovering hidden resources, exploiting vulnerabilities, and gaining elevated access to the target system.

## Key Learnings

- Nmap Scanning
- Directory Enumeration
- Web Exploitation
- File Upload Vulnerabilities
- Linux Privilege Escalation
- Reconnaissance Methodology
