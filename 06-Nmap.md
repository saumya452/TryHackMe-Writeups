# Nmap

## Summary

I learned how to use Nmap, one of the most popular network scanning tools used in cybersecurity. This room introduced host discovery, port scanning, service detection, and basic reconnaissance techniques.

## Key Learnings

- Learned the purpose of Nmap in network reconnaissance
- Performed host discovery scans
- Identified open ports on target systems
- Learned basic Nmap scan options and flags
- Understood how Nmap helps security professionals gather information about systems
