# Blue

## Summary

I completed the Blue room on TryHackMe, a beginner-friendly Windows CTF focused on enumeration, exploitation, and privilege escalation. The room provided hands-on experience with identifying vulnerable services, understanding Windows security weaknesses, and gaining administrative access.

## Key Learnings

- Windows Enumeration
- SMB Enumeration
- EternalBlue Vulnerability
- Exploitation Basics
- Privilege Escalation
- Understanding Windows Security
