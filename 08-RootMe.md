# RootMe

## Summary

I completed the RootMe room on TryHackMe, a beginner-friendly CTF focused on web exploitation and privilege escalation. The room provided hands-on experience in identifying vulnerabilities, gaining access to a target machine, and obtaining root privileges.

## Key Learnings

- Web Enumeration
- File Upload Exploitation
- Linux Enumeration
- Privilege Escalation
- Problem Solving in CTF Environments
